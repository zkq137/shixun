# MySQL 数据库使用步骤

## 1. 连接信息

```text
主机：39.107.99.7
端口：3306
数据库：talent_pipeline_platform
用户名：devuser
密码：0721
```

## 2. 先确认网络是否能连

打开 Windows PowerShell，执行：

```powershell
Test-NetConnection 39.107.99.7 -Port 3306
```

看到下面结果表示可以连接：

```text
TcpTestSucceeded : True
```

如果是 `False`，先确认自己是否连接学校软件园网络。

## 3. 使用 MySQL 命令行连接

本机安装了 MySQL 客户端后，在 PowerShell 执行：

```powershell
mysql -h 39.107.99.7 -P 3306 -u devuser -p
```

提示输入密码时输入：

```text
0721
```

密码输入时不会显示，这是正常的。

登录成功后会看到：

```text
mysql>
```

## 4. 进入项目数据库

在 `mysql>` 中执行：

```sql
USE talent_pipeline_platform;
```

查看已有表：

```sql
SHOW TABLES;
```

## 5. 验证是否可用

在 `mysql>` 中依次执行：

```sql
CREATE TABLE IF NOT EXISTS db_check (
  id INT PRIMARY KEY AUTO_INCREMENT,
  message VARCHAR(100) NOT NULL,
  created_at TIMESTAMP DEFAULT CURRENT_TIMESTAMP
);
```

```sql
INSERT INTO db_check (message) VALUES ('remote mysql ok');
```

```sql
SELECT * FROM db_check;
```

能查到 `remote mysql ok`，说明数据库可以正常使用。

## 7. DBeaver 连接方式

新建 MySQL 连接，填写：

```text
Server Host：39.107.99.7
Port：3306
Database：talent_pipeline_platform
Username：devuser
Password：0721
```

点击 `Test Connection`。如果提示下载驱动，点击下载即可。

## 8. 常见问题

### Test-NetConnection 是 False

处理方式：

```text
1. 确认连接的是学校软件园网络
2. 浏览器搜索“我的 IP”
3. 把显示的公网 IP 发给服务器管理员加入 3306 白名单
```

### Access denied for user

确认填写：

```text
用户名：devuser
密码：0721
```

### Unknown database

确认数据库名填写：

```text
talent_pipeline_platform
```

## 9. 注意

统一使用 `devuser`，不要使用 `root`。

不要随意执行：

```sql
DROP DATABASE;
DROP TABLE;
TRUNCATE TABLE;
```
